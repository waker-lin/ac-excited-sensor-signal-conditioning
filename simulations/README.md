# Simulations

Store simulation source files and exported results here.

- `multisim/`: original Multisim design files for each module and the full system
- `ltspice/`: supporting equivalent simulations if needed
- `proteus/`: alternative platform files if used later
- `exported_waveforms/`: screenshots, CSV exports, and post-processed plots

Recommended practice:

- keep one subfolder or clear filename prefix per module
- include version or date in exported waveform names
- record the operating condition in the filename or companion note
