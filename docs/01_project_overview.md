# 01 Project Overview

## 项目定位

本项目围绕一套完整的交流激励式传感器信号调理电路展开，文档组织目标是直接服务于课程设计说明书和后续扩写，而不是保留松散笔记。

## 结构约定

全文按三部分组织：

1. 总体设计思路
2. 单元电路设计、仿真与调试
3. 材料清单

这样安排的原因很直接：

- 先把系统目标、技术路线和总电路图讲清楚
- 再把每个电路单元写成闭环记录
- 最后补充器件和材料信息

## 总体部分需要回答的问题

在进入各单元之前，先说明：

- 测量目标是什么
- 为什么采用交流激励
- 为什么采用同步检波路线
- 总电路图中各模块如何对应到实际单元

## 单元电路部分的写法

每个单元统一采用以下顺序：

`电路设计 -> 参数计算 -> 器件选型 -> 仿真结果 -> 调试与实测结果`

当前单元包括：

- 正弦驱动电路
- 交流全桥及调零电路
- 三运放高共模抑制比放大电路
- 方波转换电路
- 开关式全波相敏检波电路
- 移相器
- 低通滤波器
- 直流放大电路

把仿真和调试并入各单元之后，阅读和扩写都会更直接，不会在不同章节重复同一模块。

## 材料清单部分

材料清单单独列出，不混入正文分析。内容只保留：

- 有源器件
- 无源器件
- 仪器与辅助材料

## 当前写作原则

整个仓库按下面这条规则推进：

`先整体，后单元；单元内部闭环；最后整理材料`
