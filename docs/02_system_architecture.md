# 02 System Architecture

## Positioning

This document describes the end-to-end logic of the AC-excited sensor signal-conditioning chain. The focus is not only the block order, but also why each block exists and how the signal form changes from one stage to the next.

## System Objective

The sensor front end does not directly provide a convenient DC quantity. Instead, the physical variable modulates an AC carrier generated by the excitation source. The downstream chain must therefore complete three tasks:

1. Generate a stable excitation/reference basis.
2. Extract the weak differential sensor response from a high-common-mode environment.
3. Convert the modulated AC signal into a stable signed DC output suitable for indication or further measurement.

## Main Signal Chain

`Sine-Wave Generator -> Full Bridge / Differential Transformer -> Three-Op-Amp High-CMRR Amplifier -> Square-Wave Conversion -> Phase-Sensitive Demodulator -> Low-Pass Filter -> DC Amplifier -> 3.5-Digit Display`

## Path Decomposition

Although the system is usually drawn as a single left-to-right chain, it is more accurate to understand it as two coupled paths:

- measurement path: carries the weak sensor signal that contains amplitude and phase information
- reference path: derives a timing/phase reference from the excitation source for synchronous demodulation

The final DC output is meaningful only when these two paths remain phase-consistent.

## Functional Block Description

### 1. Sine-Wave Generator

The generator establishes the carrier used to excite the sensor or inductive structure. It is the timing origin of the whole system. Its stability directly affects demodulation accuracy, because both sensor response and synchronous reference are ultimately compared against this source.

### 2. Full Bridge / Differential Transformer

This stage converts the measured physical variation into a small AC differential signal. Depending on the implementation, it may be a bridge-type transducer or a differential transformer. In either case, the useful information is not simply instantaneous voltage level; it appears as a change in AC amplitude and often also as a phase relation relative to excitation.

### 3. Three-Op-Amp High-CMRR Amplifier

The sensor output is usually weak and can ride on a considerable common-mode component or interference pickup. The three-op-amp instrumentation-amplifier style stage amplifies the differential term while suppressing common-mode content. This block determines whether the following demodulation stage receives a clean, usable representation of the sensor signal.

### 4. Square-Wave Conversion

The reference path reshapes the excitation-related sinusoid into a polarity reference, typically a square wave synchronized to the sign of the carrier. The purpose is not to preserve analog amplitude, but to mark positive and negative half-cycles so that the detector knows when the sensor output should be added or inverted.

### 5. Phase-Sensitive Demodulator

This block is the core discriminator of sign and phase. It multiplies, commutates, or polarity-switches the amplified sensor signal according to the reference path. Signals with the correct phase relationship produce a non-zero average after filtering, while orthogonal or unrelated components are strongly suppressed.

### 6. Low-Pass Filter

After demodulation, the waveform contains a desired low-frequency or DC term plus higher-frequency components generated by multiplication or switching. The low-pass filter removes the carrier-related ripple and retains the slowly varying component that represents the measurement result.

### 7. DC Amplifier

The filtered signal may already be proportional to the measured variable, but it is often too small for direct display. The DC amplifier rescales this signal into the target range for the output stage and may also provide offset trimming, zero adjustment, or final sensitivity calibration.

### 8. 3.5-Digit Display

The last stage converts the conditioned analog DC level into a readable numeric quantity. At this point, the design emphasis shifts from carrier processing to output scaling, reference stability, display range, and reading convenience.

## System Logic From Start To Finish

The architecture can be understood as a sequence of three transformations:

### A. Physical Variable -> AC Modulation

The excitation source drives the sensor, and the sensor converts the physical quantity into an AC response. This response is weak, differential, and still embedded in the carrier domain.

### B. AC Modulation -> Signed Low-Frequency Quantity

The amplifier raises the signal to a usable level, while the synchronous reference makes the sign of each half-cycle meaningful. The phase-sensitive detector then translates phase-consistent AC information into a waveform with a non-zero average.

### C. Signed Low-Frequency Quantity -> Readable DC Output

The low-pass filter extracts the average term, and the DC amplifier scales it for the display. The display therefore does not directly measure the raw sensor AC; it reports the final DC representation obtained after synchronous extraction.

## Why This Architecture Is Suitable

Compared with a direct DC measurement chain, this architecture offers several advantages for small-signal sensing:

- AC excitation helps separate the useful signal from DC drift and offset
- differential front-end amplification improves rejection of common-mode interference
- phase-sensitive demodulation preserves sign information around the balance point
- low-pass extraction produces a stable output that is easier to read and calibrate

## Interfaces Between Blocks

The system is only robust when the interfaces are treated carefully:

- Generator -> Sensor: excitation frequency and amplitude must match the sensor operating region.
- Sensor -> Amplifier: differential magnitude, source impedance, and common-mode level must stay within amplifier limits.
- Amplifier -> Demodulator: gain must be high enough for detection, but not so high that switching spikes or clipping dominate.
- Reference path -> Demodulator: phase alignment must be controlled, otherwise the output scale collapses by the `cos(phi)` factor.
- Demodulator -> Low-pass: the filter cutoff must reject carrier products while still following the measurement bandwidth.
- DC amplifier -> Display: output range, offset, and resolution must match the meter input characteristics.
