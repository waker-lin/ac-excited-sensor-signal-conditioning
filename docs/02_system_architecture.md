# 1 总体设计思路与总电路图

## 1.1 设计目标

本项目面向电阻应变式拉压力传感器应用电路设计，目标不是只得到一张原理图，而是建立一条完整的信号调理链路，使被测力信号能够完成以下转换：

- 力学量 -> 应变
- 应变 -> 电阻变化
- 电阻变化 -> 微弱交流差动电压
- 交流差动电压 -> 可滤波、可放大的直流量

结合原说明书，本系统的设计重点是：

- 传感器量程为 `0-1000 N`
- 应变片桥路输出为 `mV` 级弱信号
- 需要在较强干扰条件下完成放大、检波和后续输出
- 最终输出应具备可观测、可调试、可继续显示或数字化处理的条件

## 1.2 总体设计思路

### 1.2.1 采用交流激励的原因

原方案采用交流激励而不是直流激励，原因很明确：传感器工作环境复杂、外部干扰较大，而桥路输出又很弱。采用交流激励后，测量信息被搬移到载波上，后续可以通过相敏检波只提取与参考同相的有效分量，从而提高抗零漂和抗干扰能力。

### 1.2.2 采用这条信号链的原因

根据原说明书的方案比较，最终采用的是以下技术路线：

- `RC` 桥式正弦波振荡电路产生激励源
- 交流全桥将应变引起的电阻变化转换为差动交流信号
- 三运放高共模抑制比放大电路完成前级弱信号放大
- 过零比较器将参考正弦转换为同频方波
- 开关式全波相敏检波电路完成同步解调
- 二阶低通滤波器提取平均分量
- 直流放大电路将滤波后的低电平信号放大到可用范围

这条链路的设计思想是先保留交流相位信息，再在后级完成同步提取，而不是在桥路输出端直接做普通整流或直接放大。

### 1.2.3 关键单元选型原因

原方案对关键单元做了明确取舍，核心逻辑如下：

- 正弦驱动部分选 `RC` 桥式振荡器，是因为目标频率较高时，`LC` 振荡器所需电感、电容体积较大，结构不如 `RC` 方案简洁
- 差动放大部分选三运放结构，是因为其共模抑制能力和增益调节能力优于普通双运放差动放大方案
- 检波部分选开关式相敏检波，是因为包络检波无法区分相位，相加式相敏检波结构较复杂，而开关式方案更适合本系统的同步解调需求
- 滤波部分选二阶无限增益多路反馈低通滤波器，是因为该结构选择性较好且工作稳定，适合去除解调后的高频残余项

## 1.3 系统信号处理流程

系统信号处理流程可概括为：

`正弦驱动 -> 交流全桥 -> 三运放放大 -> 方波参考 -> 相敏检波 -> 低通滤波 -> 直流放大 -> 输出`

若按功能拆分，可分为两条相互配合的路径。

### 1.3.1 测量路径

测量路径负责传递被测信息：

`被测力 -> 弹性元件 -> 应变片 -> 交流全桥 -> 差动放大 -> 相敏检波 -> 低通滤波 -> 直流放大`

这一路中的关键信息是桥路输出信号的幅值和相位。前级放大必须尽量保留这两部分信息，否则后面的同步解调就失去意义。

### 1.3.2 参考路径

参考路径负责提供同步判据：

`正弦激励 -> 方波转换 -> 相敏检波控制端`

这一路不承担测量幅值，只负责给出同频参考和相位基准。只要参考路径的频率和相位关系稳定，相敏检波输出的平均值就能正确反映被测信号的同相分量。

### 1.3.3 信号状态演化

沿着主链路，信号状态经历了三次本质变化：

- 在桥路处，机械量被变换为微弱交流差动信号
- 在相敏检波处，交流相位信息被变换为平均值极性信息
- 在低通和直流放大后，平均值被变换为可直接读取的直流量

因此，本系统不是简单的“放大电压”，而是在完成一次从交流调制量到直流表示量的转换。

### 1.4.2 总体电路图

![Overall circuit simulation](../images/overall_circuit_simulation.png)

从总电路图可以直接看出，整个模拟调理部分由以下模块串联构成：

- 正弦波发生器
- 全桥电路
- 三运放高共模抑制比放大电路
- 方波转换电路
- 开关式相敏检波电路
- 低通滤波器
- 直流放大器

后续各单元分析都沿着这条主链路展开。

### 1.4.3 总电路图阅读方式

阅读总电路图时，应按“激励源、桥路、放大、参考整形、同步检波、滤波、直流放大”的顺序理解，而不是把它当作若干无关模块并列堆放。

其中有三点最关键：

- 正弦波发生器既为桥路提供交流激励，也为参考路径提供频率基准
- 方波转换电路并不处理测量幅值，只负责为相敏检波提供同步控制信号
- 相敏检波、低通滤波和直流放大三者共同完成“交流测量量 -> 直流输出量”的最终转换

### 1.4.4 本文聚焦范围

原课程设计流程图中还包含 `A/D` 转换、微处理器和数码显示部分，本文先聚焦模拟信号调理主链路，即从正弦驱动到直流放大这一段。

原因很简单：这一段决定了测量精度、抗干扰能力和最终输出极性的正确性，是整个系统最核心的电路部分。
